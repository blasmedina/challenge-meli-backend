# Challenge Meli Backend

Challenge para Mercado Libre por Blas Medina. Api REST.

## Comenzando

### Pre-requisitos

- Node: v10 para arriba

### Instalación

```bash
npm install
```

```bash
npm run build
```

```bash
npm run start
```

Y para ejecutar en ambiente de desarrollo...

```bash
npm run dev
```

## Prueba

```bash
npm run test
```

### Verificación de codificación

```bash
npm run lint
```

## Construido con

- [Express](https://expressjs.com/)
